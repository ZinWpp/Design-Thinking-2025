# 🌊 Coral Heat Stress Monitor
**ระบบติดตามอุณหภูมิผิวน้ำทะเลและคำนวณความเสี่ยงปะการังฟอกขาวในพื้นที่หลัก** 
แอปพลิเคชันนี้ใช้สำหรับเฝ้าระวังปรากฏการณ์ปะการังฟอกขาว โดยการสกัดข้อมูลอุณหภูมิผิวน้ำทะเล (Sea Surface Temperature) จากแหล่งข้อมูลที่น่าเชื่อถือผ่าน API เพื่อวิเคราะห์ความเสี่ยงในแนวปะการังหลักของประเทศไทย ทั้งฝั่งอันดามันและอ่าวไทย

## 🚀 ฟีเจอร์หลัก (Features)
- **Real-time Data Extraction**: ดึงข้อมูลอุณหภูมิน้ำทะเลล่าสุดผ่าน Marine API
- **Risk Assessment**: ระบบคำนวณความเสี่ยงฟอกขาวอัตโนมัติตามเกณฑ์อุณหภูมิสะสม 
- **Interactive Visualization**: แสดงกราฟแนวโน้มอุณหภูมิย้อนหลังและแผนที่จุดพิกัดที่สำรวจ
- **Data Export**: สามารถบันทึกและดาวน์โหลดข้อมูลที่สกัดได้เป็นไฟล์ CSV
